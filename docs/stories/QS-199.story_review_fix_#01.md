### [must-fix] M6 — cache-buster regex misses shared-to-shared imports
- File: `custom_components/quiet_solar/ui/dashboard.py:289` (`_IMPORT_REWRITE_RE`)
- Severity: must-fix
- Source: qs-review-edge-case-hunter
- Description: The regex only matches `./shared/…` prefixes. Sibling
  imports *inside* `shared/` use `./qs-card-base.js` (no `./shared/`
  prefix) — e.g. `shared/qs-ring-duration-base.js` importing
  `./qs-card-base.js`. Those are never cache-busted, so after a restart the
  browser loads the card tagged, then resolves the inheritance chain
  through an **untagged** `./qs-card-base.js` that it caches forever. Future
  edits to `qs-card-base.js` won't propagate for any card extending
  `QsRingDurationCardBase`; two copies of `QsCardBase` also get loaded.
- Proposed fix: Widen the regex to match the optional prefix (e.g.
  `\./(?:shared/)?[^'"?\s]+\.js`) so sibling imports inside `shared/` are
  also rewritten. Ensure the rewrite actually fires on `shared/*` files
  (it iterates them but currently no match fires). Add a test that a
  `shared/` file's `./sibling.js` import gets the `?qs_tag=` buster.

### [must-fix] M7 — UnicodeDecodeError aborts resource copy mid-recursion
- File: `custom_components/quiet_solar/ui/dashboard.py:377`
  (`_async_copy_and_register_resources`)
- Severity: must-fix
- Source: qs-review-edge-case-hunter
- Description: The JS path now does `raw.decode("utf-8")`. A `.js` file with
  non-UTF-8 bytes (Latin-1, BOM mojibake) raises `UnicodeDecodeError`, which
  bubbles to `async_update_resources`' generic `except Exception`, logs a
  warning, and aborts mid-recursion → destination tree partially
  copied/registered, leaving stale/missing cards. Pre-QS-199 used pure
  `rb`/`wb` so any encoding round-tripped.
- Proposed fix: Only decode/rewrite files that need import rewriting, and
  guard the decode: on `UnicodeDecodeError` fall back to a raw byte copy
  (no rewrite) instead of aborting. Add a test with a non-UTF-8 `.js`
  fixture asserting the copy completes and other resources still register.
